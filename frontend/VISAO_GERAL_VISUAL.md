## 🚀 Como Usar

### 1️⃣ Instalação

```bash
cd E:\VSCode\Projects\UltraSinger\frontend
npm install
```

### 2️⃣ Desenvolvimento

```bash
npm run dev
```

Acesse: **http://localhost:3000**

### 3️⃣ Build para Produção

```bash
npm run build
```

Arquivos otimizados em `dist/`

### 4️⃣ Preview da Build

```bash
npm run preview
```

## 💡 Dicas Importantes

1. **Sempre rode `npm install` antes de iniciar**
2. **O backend deve rodar na porta 5000** (proxy configurado)
3. **Use `npm run dev` para desenvolvimento** (hot reload)
4. **Build de produção com `npm run build`**
5. **Verifique o console do navegador para erros**
6. **Todos os componentes são reutilizáveis**
7. **Design system totalmente customizável no `tailwind.config.js`**
